## Batch Prompts (gamma.app) — 3 Akte, 14 Kapitel

### Nutzung
- Pro Asset 1 Bild generieren (Format siehe `target_aspect_ratio` in CSV).
- Benenne Export-Dateien **exakt** wie in `assets/manifest_assets.csv` unter `output_path_print`.
- Wenn ein Bild „matschig“ wirkt: **Kontrast erhöhen + Details/Textures explizit verlangen**, keine Text-Overlays.
- **DETAIL**-Assets: Format 4:3 (für Spaltenbreite).
- **TEXTURE**-Assets: Format 1:1 (tileable, für Overlay).
- **HERO**-Assets: Format 17:22 (Hochkant 8.5x11).

## Napkin.ai Batch (Infografiken)

Alle Infografiken siehe `assets/manifest_assets.csv` Spalte `prompt` (Kategorie `infographic`, `map`, `diagram`).

**Wichtig für napkin.ai:**
- Export als **Vektor (SVG/PDF)** bevorzugt, sonst **300dpi PNG**.
- **Sehr große Labels** (min. 12–16pt für Haupttexte).
- **Viel Weißraum**, keine ausgefransten Kanten.
- **2–3 Farben max** (siehe Styleguide pro Akt).
