# Typografie-Analyse: Magazin-Ästhetik

## Aktuelle Situation

### Spaltenmaße
- **Textbereich**: 6.7" (170mm)
- **Spalte 1**: 30% = 2.01" (51mm) → ~40-45 Zeichen/Zeile
- **Spalte 2**: 30% = 2.01" (51mm) → ~40-45 Zeichen/Zeile
- **Spalte 3**: 26% = 1.74" (44mm) → ~35-40 Zeichen/Zeile
- **Sidebar**: 14% = 0.94" (24mm)
- **Spaltenabstand**: 0.18" (4.6mm)

### Typografie
- **Schriftgröße**: 11pt
- **Zeilenabstand**: 1.0x (11pt) - Standard
- **Kein Grundlinienraster**

## Probleme

### ❌ Spalten zu schmal
- **Optimal**: 50-75 Zeichen pro Zeile für 11pt
- **Aktuell**: 35-45 Zeichen → **zu eng!**
- Folge: Text wirkt gedrängt, Lesbarkeit leidet

### ❌ Zeilenabstand zu eng
- **Optimal für Magazin**: 1.2-1.5x (13-16pt bei 11pt)
- **Aktuell**: 1.0x (11pt) → **zu kompakt!**
- Folge: Text wirkt dicht, wenig Luft

### ❌ Spaltenabstand zu eng
- **Optimal für Magazin**: 0.25-0.3" (6-8mm)
- **Aktuell**: 0.18" (4.6mm) → **zu eng!**
- Folge: Spalten wirken zusammengedrängt

## Empfehlungen

### Option A: 2 gleich breite Spalten (klassisch, lesbar)
- Spalte 1: 48% = 3.22" (82mm) → ~55 Zeichen
- Spalte 2: 48% = 3.22" (82mm) → ~55 Zeichen
- Sidebar: 4% = 0.26" (7mm)

### Option B: Asymmetrisch (Magazin-typisch)
- Spalte 1: 40% = 2.68" (68mm) → ~60 Zeichen
- Spalte 2: 35% = 2.35" (60mm) → ~52 Zeichen
- Spalte 3: 19% = 1.27" (32mm) → ~28 Zeichen (für Pullquotes)
- Sidebar: 6% = 0.40" (10mm)

### Typografie-Verbesserungen
1. Zeilenabstand: 1.3x (14.3pt)
2. Spaltenabstand: 0.25" (6.4mm)
3. Grundlinienraster implementieren
