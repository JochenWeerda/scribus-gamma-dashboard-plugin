# Gamma Scribus Pack

- Typ: Scribus Script Addon (Python)
- Kompatibel: Scribus 1.6.4 stable (Windows 11 getestet)
- Zweck: Gamma ZIP/PPTX importieren, Cards clustern, Crops + Hints erzeugen, Setzerei-Engine versorgen
- Install: `install.ps1` oder manuell `pip install -r requirements.txt`
- Proof: `proof_deps.ps1` (Import-Check)
- Konfiguration: `ZC_GAMMA_EXPORT_DIR`, `ZC_PROJECT_DIR`, `ZC_VENV_PY`, optional `ZC_GAMMA_HINTS`
